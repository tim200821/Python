# Python
Работы на Python за 1-4 курс

Convert -- этот файл преобразует число с помощью функций, в нем используется модель MVP(Model View Presentor), данная программа преобразует число (Bin, Dec, Hex) в число (Bin, Dec, Hex).

MVP in Python -- программа записывающая имя и возрост выброного животного и выводят их при запросе, также в этой программе использованна модель MVP(Model View Presentor).

Classes -- в этой папке несколько файлов рассматривающих работу и применение классов в Python.

Данные работы в открытом доступе для взоимствования и дальнейшего использования!
